# Riot! Riot! — Site oficial

Site de página única da banda Riot! Riot! (cover de Paramore, Floripa).
Hospedado no Cloudflare Pages: https://riotriot.pages.dev

## Editar
Tudo é estático. Edite `index.html` (conteúdo), `styles.css` (visual) e
`main.js` (menu mobile/scroll). Sem build. Faça commit/push e o Cloudflare publica.

## Pré-visualizar localmente
    python3 -m http.server 8000
    # abra http://localhost:8000
